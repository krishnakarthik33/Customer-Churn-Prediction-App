📊 Customer Churn Prediction App
A machine learning-powered web application developed using Streamlit that predicts whether a customer is likely to churn based on customer demographics, banking activity, and account information.

The application provides real-time churn prediction with an interactive and user-friendly dashboard interface.

The project integrates trained machine learning models with data preprocessing techniques such as scaling and encoding to generate accurate predictions.

It also includes data visualization features like churn probability gauge charts and feature importance analysis for better business insights.

🚀 Features
Real-time customer churn prediction
Interactive web interface using Streamlit
Machine learning model integration
Churn probability visualization with Plotly gauge charts
Feature importance analysis
Dynamic customer input handling
Data preprocessing using encoders and scalers
